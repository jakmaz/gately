# Web Editor (app.gately.dev)

React SPA serving the Gately logic gate editor.

## Tech Stack

- Vite + React 19 + TypeScript
- Tailwind v4
- @gately/ui components
- @gately/core simulator

## Development

```bash
bun install
bun dev        # localhost:5173
bun build      # Build for production
```

## Deployment

Deploys to Cloudflare Pages at app.gately.dev
